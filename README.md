Reservation System — система бронирования
RESTful веб-приложение на Java + Spring Boot. 
Реализована многослойная архитектура: контроллеры для обработки HTTP-запросов, сервисный слой с бизнес-логикой, работа с PostgreSQL через JPA/Hibernate. 
Настроен глобальный обработчик ошибок для единого формата ответов API. 
Приложение упаковано в Docker для простого развёртывания и тестирования.
